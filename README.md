# Clothes Cluster
紡拓會 - 衣著分群
## 計畫目標:

以基於深度學習方法對服裝影像擷取Color、Material、Texture特徵，並用於分群。
創建使用者介面，讓使用者選擇要分群的目錄，透過模型對影像分析並輸出分群後結果。
